22A Shared Risk

90Prime Upgrade: Information for 22A Shared Risk Science During the second half of 22A, you will have the option of using the newly updated 90Prime for you science on a shared risk basis. Here we provide some basic information.

If you are interested, please contact David Sand (dsand@arizona.edu). As the upgraded 90Prime is being developed with a separate dewar, you can also choose to use the current 90Prime as well. Scheduling dewar changes between the two systems will be handled by Mountain Operations, and will be worked into the telescope schedule beforehand.

While we anticipate that the newly upgraded 90Prime will be available for shared risk science in the second half of the 22A semester, many details are still be worked out as the new focal plane is designed and constructed. Here is what you can expect:

  • A new focal plane and CCD set that should be cosmetically excellent and have at least the same QE as the current 90Prime. The chips are blue sensitive, just as the current set is.
  • The chip gaps will be roughly the same or smaller, but otherwise the field of view, pixel size, etc should be nearly the same as the current 90Prime.
  • The readout time will be significantly improved, roughly half of the current 30 seconds it takes to read and write a full frame image.
  • There will be new user GUIs (and backend control software) which will hopefully make the operation of 90Prime easier and less prone to fatal ‘hanging’ errors.
  • Other components will be replaced and upgraded, including a new dewar window, shutter, filter wheel components and a new spring mechanism for focus.
  • Preliminary tests have shown that the new springs improve focus drift during longer integrations. Nonetheless, we plan on implementing a ‘focus maintenance’ code that will both nudge the focus when necessary, and adjust the focus when slewing to a new field. As this has not been tested at all as of yet, it is unlikely this will be implemented in 22A.

About

90Prime is a prime focus wide-field imager for the 90-inch telescope located at Kitt Peak. The camera’s focal plane array is populated with a mosaic of four thinned Lockheed 4096 × 4096 pixel CCDs. The f/2.98 system provides a plate scale of 0 0.45 arcsec/pixel and a total field-of-view of 1 .16° × 1.16°. The optical design includes a four element corrector and six position filter wheel. The first science run was conducted in November 2003.
